# Bitwise Operators

## Operators

- Binary AND: `&`

```python
a & b
```

- Binary OR: `|`

```python
a | b
```

- Binary XOR: `^`

```python
a ^ b
```

1 ^ 1 = 0
1 ^ 0 = 1

- Binary NOT: `~`

```python
~ a
```

- Binary Left Shift: `<<`

```python
a<<
```

- Binary Right Shift: `>>`

```python
a>>
```
