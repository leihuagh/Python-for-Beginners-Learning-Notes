# Comparison

## Usages

- Equal To: `==`

```python
a == b
```

- Not Equal To: `!=`

```python
a != b
```

- Greater Than: `>`

```python
a > b
```

- Less Than: `<`

```python
a < b
```

- Greater Than Equal To: `>=`

```python
a >= b
```

- Less Than Equal To: `<=`

```python
a <= b
```
