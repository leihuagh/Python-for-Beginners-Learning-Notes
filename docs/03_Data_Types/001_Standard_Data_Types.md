# Standard Data Types

## Types

- Values cannot be changed (Immutable Data Types)
- Values can be changed (Mutable Data Types)

## Immutable Data Types

- Numbers
    - Integers
    - Floating
- Strings
- Tuples
    - Using `()`

## Mutable Data Types

- Lists
    - Using `[]`
- Dictionaries
    - Key-Value pair
- Sets
